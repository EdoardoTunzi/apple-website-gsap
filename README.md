# Apple-Inspired 3D Website

Per fare pratica con GSAP e Three.js, partendo da un tutorial ho ricreato questo sito web moderno ispirato al design Apple, realizzato con React, Three.js, GSAP, Tailwind CSS e Zustand. 

Il progetto offre un'esperienza immersiva grazie a modelli 3D interattivi, animazioni sincronizzate allo scroll e sezioni dinamiche con effetti visivi fluidi.

Questo progetto rappresenta un esempio di sviluppo frontend moderno, combinando grafica 3D, animazioni ad alte prestazioni e un design curato per creare un'esperienza utente coinvolgente.

## Caratteristiche

- Modelli 3D interattivi con Three.js
- Animazioni avanzate basate sullo scroll tramite GSAP
- Sezioni "pinned" per una navigazione coinvolgente
- Transizioni fluide tra immagini e contenuti
- Design responsive ottimizzato per desktop e mobile
- Interfaccia moderna ispirata all'estetica Apple

## Tecnologie utilizzate

- React Vite
- Typescript
- Three.js
- GSAP
- Tailwind CSS
- Zustand
- clsx
- react-responsive
